World Map

Map tile pattern for embedding: /public_files/pi/marblemaps/earth/opentopomap/{z}/{x}/{y}.png

Maps are stored in /home/pi/.local/share/marble/maps/, and made publically viewable by a bind mount. This takes advantage of the fact that KDE marble uses a very simple on-disk format.

This app does not automatically fetch maps from any internet service and only shows what is on the server.

To update or get new maps, either browse around using Marble, or rsync from /home/USERNAME/.local/share/marble/maps/ on a real computer.
